# Examples

These modules demonstrate alternative animations built on top of the base data structures defined in `src/js/tree-layout.js`.

| Example               | Description |
|-----------------------|-------------|
| `lightning_flash.js`  | Animates a particle down the lightning path (Kether → Binah → Gevurah → Tiphareth → Malkuth). |
| `serpent_path.js`     | Animates a particle along a winding serpent path that climbs and descends the tree. |
| `planetary_hours.js`  | Modulates the glow of each sephira by its associated planetary hour. |

To try an example, copy its code into a `<script type="module">` tag in `index.html` in place of `main.js`, or adapt it into your own sketch.
